Lesson 9: File Handling

Read, write, and manage files in Python

Introduction

File handling is essential for working with data. Python makes it easy to read, write, and manipulate files.

Opening Files

Use the open() function with file mode:

file = open("filename.txt", "r")  # Read
file = open("filename.txt", "w")  # Write
file = open("filename.txt", "a")  # Append
file = open("filename.txt", "x")  # Create

Reading Files

with open("file.txt", "r") as file:
    content = file.read()         # Read entire file
    lines = file.readlines()      # Read as list
    line = file.readline()        # Read one line

Writing to Files

with open("file.txt", "w") as file:
    file.write("Hello World\n")
    file.writelines(["Line 1\n", "Line 2\n"])

Best Practices

  • Always use with statement to auto-close files
  • Use appropriate file modes
  • Handle file exceptions
  • Test file existence before reading
  • Close files properly

File Operations

import os
os.remove("file.txt")        # Delete file
os.rename("old.txt", "new.txt")  # Rename
if os.path.exists("file.txt"):   # Check existence
    print("File exists")
