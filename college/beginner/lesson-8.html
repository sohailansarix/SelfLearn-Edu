Dictionaries and Sets

25 minutes • Beginner

Learning Objectives

  • Create and use dictionaries
  • Understand dictionary methods
  • Work with sets
  • Use set operations

Dictionaries

Dictionaries store key-value pairs in curly braces.

Creating Dictionaries

my_dict = {"name": "Alice", "age": 25, "city": "NYC"}
student = {1: "Alice", 2: "Bob", 3: "Charlie"}

Accessing Dictionary Values

person = {"name": "Alice", "age": 25}
print(person["name"])      # Alice
print(person.get("age"))   # 25
print(person.get("city", "Unknown"))  # Unknown

Dictionary Methods

person = {"name": "Alice", "age": 25}
person["city"] = "NYC"     # Add
del person["age"]          # Delete
keys = person.keys()       # Get keys
values = person.values()   # Get values
items = person.items()     # Get pairs

Sets

Sets store unique, unordered values.

Creating Sets

my_set = {1, 2, 3, 4, 5}
fruits = {"apple", "banana", "cherry"}
empty_set = set()  # Use set(), not {}

Set Operations

set_a = {1, 2, 3}
set_b = {3, 4, 5}
print(set_a.union(set_b))        # {1, 2, 3, 4, 5}
print(set_a.intersection(set_b)) # {3}
print(set_a.difference(set_b))   # {1, 2}

Practice Exercises

Exercise 8.1

Create a dictionary of student grades and find the average.

Solution
grades = {"Alice": 90, "Bob": 85, "Charlie": 95}
average = sum(grades.values()) / len(grades)
print(average)
