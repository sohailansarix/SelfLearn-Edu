Control Flow - If/Else Statements

20 minutes • Beginner

Learning Objectives

  • Understand conditional statements (if, elif, else)
  • Use comparison and logical operators
  • Write nested conditional statements
  • Apply if/else statements to solve problems

What are Conditional Statements?

Conditional statements allow your program to make decisions and execute different code based on conditions.

The if Statement

if condition:
    # code executes if condition is True
    print("Condition is True")

Example: if Statement

age = 18
if age >= 18:
    print("You are an adult")

Output: You are an adult

The if...else Statement

The else clause provides code to execute when the condition is False.

Example: if...else Statement

score = 75
if score >= 75:
    print("Pass")
else:
    print("Fail")

The if...elif...else Statement

Use elif to check multiple conditions.

Example: if...elif...else

age = 15
if age < 13:
    print("Child")
elif age < 18:
    print("Teenager")
else:
    print("Adult")

Comparison Operators

  • == (equal to)
  • != (not equal to)
  • > (greater than)
  • < (less than)
  • >= (greater than or equal)
  • <= (less than or equal)

Logical Operators

  • and - Both conditions must be True
  • or - At least one condition must be True
  • not - Reverses the boolean result

Example: Logical Operators

age = 25
has_license = True
if age >= 18 and has_license:
    print("You can drive")

Practice Exercises

Exercise 4.1

Write a program that checks if a number is positive, negative, or zero.

Solution 
num = int(input("Enter a number: "))
if num > 0:
    print("Positive")
elif num < 0:
    print("Negative")
else:
    print("Zero")

Exercise 4.2

Create a program that determines if a person can vote (age >= 18).

Solution 
age = int(input("Enter your age: "))
if age >= 18:
    print("You can vote")
else:
    print("You cannot vote yet")
