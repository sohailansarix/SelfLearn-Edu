Lesson 4: Control Flow - If/Else Statements

Learn how to make decisions in Python

Introduction

Control flow statements allow you to execute different blocks of code based on certain conditions. The most fundamental control flow structure is the if/else statement.

The if Statement

An if statement evaluates a condition and executes code only if that condition is True.

if condition:
    # Code executes if condition is True
    print("Condition is True")

Example: Simple if Statement

age = 18
if age >= 18:
    print("You are an adult")

Output: You are an adult

The if...else Statement

The else clause provides an alternative code block to execute if the condition is False.

if condition:
    # Code if condition is True
else:
    # Code if condition is False

Example: if...else Statement

score = 75
if score >= 75:
    print("A")
else:
    print("Not A")

Output: A

The if...elif...else Statement

The elif (else if) statement allows you to check multiple conditions.

if condition1:
    # Code if condition1 is True
elif condition2:
    # Code if condition2 is True
else:
    # Code if both are False

Example: if...elif...else Statement

age = 15
if age < 13:
    print("Child")
elif age < 18:
    print("Teenager")
else:
    print("Adult")

Output: Teenager

Comparison Operators

These operators are used in conditions:

  • == (equal to): a == b
  • != (not equal to): a != b
  • > (greater than): a > b
  • < (less than): a < b
  • >= (greater than or equal to): a >= b
  • <= (less than or equal to): a <= b

Logical Operators

Combine multiple conditions using logical operators:

  • and: Both conditions must be True
  • or: At least one condition must be True
  • not: Reverses the boolean result

Example: Using Logical Operators

age = 25
has_license = True

if age >= 18 and has_license:
    print("You can drive")
else:
    print("You cannot drive")

Output: You can drive

Nested if Statements

You can nest if statements inside each other:

if condition1:
    if condition2:
        # Code executes only if both are True
        print("Both conditions are True")

Example: Nested if Statements

age = 20
has_money = True

if age >= 18:
    if has_money:
        print("You can buy a ticket")
    else:
        print("You need money")
else:
    print("Too young")

Key Points to Remember

  • Indentation is crucial in Python - it defines code blocks
  • Conditions must evaluate to True or False
  • Use comparison operators to create conditions
  • Combine conditions with and, or, not
  • You can nest if statements for more complex logic

Practice Exercise

Hint: Use if...elif...else statements with comparison operators.
