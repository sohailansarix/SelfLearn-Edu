Lesson 7: Lists and Tuples

Master Python's most useful data structures

Introduction

Lists and tuples are sequence data types that store multiple values. They're fundamental to Python programming and are used extensively for data manipulation.

Lists

A list is an ordered, mutable (changeable) collection of items. Lists are created using square brackets [].

my_list = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.14, True]

Accessing List Elements

fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # Output: apple
print(fruits[1])    # Output: banana
print(fruits[-1])   # Output: cherry (last element)
print(fruits[0:2])  # Output: ['apple', 'banana'] (slicing)

List Methods

Method Description Example
append() Adds an element to the end list.append(4)
insert() Inserts at a specific position list.insert(1, "x")
remove() Removes the first occurrence list.remove(2)
pop() Removes and returns an element list.pop(0)
sort() Sorts the list in place list.sort()
reverse() Reverses the list in place list.reverse()

Example: List Methods

numbers = [3, 1, 4, 1, 5]
numbers.append(9)      # [3, 1, 4, 1, 5, 9]
numbers.sort()         # [1, 1, 3, 4, 5, 9]
numbers.reverse()      # [9, 5, 4, 3, 1, 1]
numbers.remove(1)      # [9, 5, 4, 3, 1]

List Comprehension

A concise way to create lists based on existing lists:

# Create a list of squares
squares = [x**2 for x in range(5)]
print(squares)  # Output: [0, 1, 4, 9, 16]

# Create a list of even numbers
evens = [x for x in range(10) if x % 2 == 0]
print(evens)    # Output: [0, 2, 4, 6, 8]

Tuples

A tuple is an ordered, immutable (unchangeable) collection of items. Tuples are created using parentheses ().

my_tuple = (1, 2, 3, 4, 5)
fruits_tuple = ("apple", "banana", "cherry")
single_item = (1,)  # Note the comma for single item

Accessing Tuple Elements

coordinates = (10, 20, 30)
print(coordinates[0])    # Output: 10
print(coordinates[1:3])  # Output: (20, 30)
print(len(coordinates))  # Output: 3

Tuple Methods

Tuples have fewer methods than lists since they're immutable:

numbers = (1, 2, 3, 2, 4, 2)
print(numbers.count(2))   # Output: 3 (count occurrences)
print(numbers.index(3))   # Output: 2 (find index)

Lists vs Tuples

Feature List Tuple
Mutable Yes No
Syntax [] ()
Speed Slower Faster
Use Case Changeable data Fixed, protected data

Unpacking

Extract values from lists or tuples into separate variables:

coordinates = (10, 20)
x, y = coordinates
print(x, y)  # Output: 10 20

# Unpacking lists
values = [1, 2, 3]
a, b, c = values
print(a, b, c)  # Output: 1 2 3

Iteration

# Iterate through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Iterate with index
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

Key Points to Remember

  • Lists are mutable and use square brackets
  • Tuples are immutable and use parentheses
  • Both support indexing and slicing
  • Use lists when you need to modify data
  • Use tuples for fixed, protected data
  • List comprehensions provide elegant list creation
  • Negative indices count from the end
