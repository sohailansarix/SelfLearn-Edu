Module 2 • Intermediate

Lists and Tuples

25 minutes • Beginner

Learning Objectives

  • Create and manipulate lists
  • Understand list methods
  • Work with tuples
  • Use indexing and slicing

Lists

Lists are mutable collections of items stored in square brackets.

Creating Lists

my_list = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.14, True]

Accessing Elements

fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # apple
print(fruits[-1])   # cherry
print(fruits[0:2])  # ["apple", "banana"]

List Methods

my_list = [1, 2, 3]
my_list.append(4)       # [1, 2, 3, 4]
my_list.insert(0, 0)    # [0, 1, 2, 3, 4]
my_list.remove(2)       # [0, 1, 3, 4]
my_list.sort()          # Sorts in place
my_list.reverse()       # Reverses in place

Tuples

Tuples are immutable sequences stored in parentheses.

Creating Tuples

my_tuple = (1, 2, 3, 4, 5)
fruits_tuple = ("apple", "banana", "cherry")
single = (1,)  # Note the comma

Tuple Operations

my_tuple = (1, 2, 3)
print(my_tuple[0])      # 1
print(my_tuple[1:3])    # (2, 3)
print(len(my_tuple))    # 3

Iteration

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

Practice Exercises

Exercise 7.1

Create a list of numbers and find the sum.

Solution
numbers = [1, 2, 3, 4, 5]
total = sum(numbers)
print(total)  # 15
