Lesson 2: Variables & Data Types

Learn how to store and work with different types of data in Python.

What are Variables?

A variable is a container for storing data values. In Python, you don't need to declare the variable type:

name = "Alice"
age = 25
height = 5.6
is_student = True

Naming Rules

  • Must start with a letter or underscore (_)
  • Can contain letters, numbers, and underscores
  • Are case-sensitive (name != Name)
  • Avoid Python keywords as variable names

Data Types

Python has several built-in data types:

1. Integers (int)

Whole numbers, positive or negative:

age = 25
temperature = -5
count = 100

2. Floats (float)

Numbers with decimal points:

price = 19.99
height = 5.6
pi = 3.14159

3. Strings (str)

Text data, enclosed in quotes:

city = 'Mumbai'
name = "Alice"
message = '''This is a
multi-line string'''

4. Booleans (bool)

True or False values:

is_logged_in = True
is_admin = False

Type Conversion (Casting)

Convert one data type to another:

age = int("25")          # "25" string to 25 integer
age_str = str(30)       # 30 integer to "30" string
price = float("19.99")  # "19.99" string to 19.99 float

Checking Data Types

Use the type() function to check the data type:

name = "Alice"
age = 25
height = 5.6
is_student = True

print(type(name))        # 
print(type(age))         # 
print(type(height))      # 
print(type(is_student))  #

Key Points to Remember

  • Variables store data values
  • Python automatically determines the data type
  • Strings need quotes (single or double)
  • Use type() to check a variable's data type
  • You can convert between data types when needed
  • Choose meaningful variable names
