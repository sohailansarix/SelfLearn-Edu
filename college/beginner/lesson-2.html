What are Variables?

A variable is a container for storing data values. In Python, you don't need to declare the variable type:

name = "Alice"
age = 25
height = 5.6
is_student = True

Naming Rules

  • Must start with letter or underscore (_)
  • Can contain letters, numbers, underscores
  • Case-sensitive (name â‰  Name)
  • Use snake_case for multiple words

Data Types

1. String (str)

name = "John"
city = 'Mumbai'

2. Integer (int)

age = 25
count = -100

3. Float (float)

price = 19.99
temperature = -5.5

4. Boolean (bool)

is_logged_in = True
is_admin = False

Type Conversion

age = int("25")
age_str = str(30)
price = float("19.99")
