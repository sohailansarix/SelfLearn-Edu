Learning Objectives

  • Understand variables and naming conventions
  • Learn Python's basic data types: int, float, str, bool
  • Perform type conversion (casting)
  • Check variable types using type()

What are Variables?

A variable is a container for storing data values. In Python, you don't need to declare the variable type:

name = "Alice"
age = 25
height = 5.6
is_student = True

Naming Rules

  • Must start with a letter or underscore (_)
  • Can contain letters, numbers, and underscores
  • Are case-sensitive (name != Name)
  • Avoid Python keywords as variable names

Data Types

Python has several built-in data types:

1. Integers (int)

Whole numbers, positive or negative:

age = 25
temperature = -5
count = 100

2. Floats (float)

Numbers with decimal points:

price = 19.99
height = 5.6
pi = 3.14159

3. Strings (str)

Text data, enclosed in quotes:

city = 'Mumbai'
name = "Alice"
message = '''This is a
multi-line string'''

4. Booleans (bool)

True or False values:

is_logged_in = True
is_admin = False

Type Conversion (Casting)

Convert one data type to another:

age = int("25")          # "25" string to 25 integer
age_str = str(30)       # 30 integer to "30" string
price = float("19.99")  # "19.99" string to 19.99 float

Checking Data Types

Use the type() function to check the data type:

name = "Alice"
age = 25
height = 5.6
is_student = True

print(type(name))        # <class 'str'>
print(type(age))         # <class 'int'>
print(type(height))      # <class 'float'>
print(type(is_student))  # <class 'bool'>

Practice Exercises

Exercise 2.2

Convert a string "100" to an integer and add 50 to it.

Solution 
num = int("100")
result = num + 50
print(result)  # Output: 150
