Lesson 6: Functions

Learn to write reusable code with functions

Introduction

A function is a reusable block of code that performs a specific task. Functions help you organize your code, avoid repetition, and make your programs more maintainable.

Defining a Function

Use the def keyword to define a function:

def function_name(parameters):
    """Docstring - explains what the function does"""
    # Function body
    return result

Example: Simple Function

def greet(name):
    """Greet a person by name"""
    return f"Hello, {name}!"

print(greet("Alice"))  # Output: Hello, Alice!

Function Parameters

Parameters are variables that receive values when the function is called. Values passed to a function are called arguments.

def add(a, b):  # a and b are parameters
    return a + b

result = add(5, 3)  # 5 and 3 are arguments

Example: Multiple Parameters

def calculate_area(length, width):
    """Calculate the area of a rectangle"""
    area = length * width
    return area

print(calculate_area(10, 5))  # Output: 50

Return Values

Use the return statement to return a value from a function. If no return statement is present, the function returns None.

def square(x):
    return x * x

result = square(4)
print(result)  # Output: 16

Default Parameters

You can provide default values for parameters:

def greet(name="Friend"):
    return f"Hello, {name}!"

print(greet())         # Output: Hello, Friend!
print(greet("Bob"))    # Output: Hello, Bob!

Keyword Arguments

You can pass arguments using parameter names:

def describe_person(name, age, city):
    return f"{name} is {age} years old and lives in {city}"

# Positional arguments
print(describe_person("Alice", 25, "NYC"))

# Keyword arguments
print(describe_person(name="Bob", age=30, city="LA"))

Variable-Length Arguments

Use *args for variable number of positional arguments and **kwargs for keyword arguments:

def sum_all(*args):
    total = 0
    for num in args:
        total += num
    return total

print(sum_all(1, 2, 3, 4, 5))  # Output: 15

Example: **kwargs

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="NYC")

Scope and Local Variables

Variables defined inside a function have local scope and only exist within that function:

x = 10  # Global variable

def my_function():
    x = 5  # Local variable
    print(x)  # Output: 5

my_function()
print(x)  # Output: 10

Docstrings

A docstring is a string at the beginning of a function that explains what it does:

def factorial(n):
    """Calculate the factorial of n"""
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial.__doc__)  # Prints the docstring

Lambda Functions

Lambda functions are anonymous functions written in a single line using the lambda keyword:

square = lambda x: x * x
print(square(5))  # Output: 25

# Lambda with multiple parameters
add = lambda x, y: x + y
print(add(3, 4))  # Output: 7

Key Points to Remember

  • Functions make code reusable and organized
  • Use def to define a function
  • Parameters are inputs, arguments are the actual values
  • Use return to send a value back
  • Provide default parameter values when appropriate
  • Use *args for variable positional arguments
  • Use **kwargs for variable keyword arguments
  • Lambda functions are useful for short, simple operations
