Functions

30 minutes • Beginner

Learning Objectives

  • Define and call functions
  • Understand parameters and arguments
  • Use return statements
  • Create reusable code with functions

What are Functions?

A function is a reusable block of code that performs a specific task.

Defining a Function

def function_name(parameters):
    """Docstring explaining what the function does"""
    # function body
    return result

Calling a Function

result = function_name(arguments)

Function Parameters and Arguments

Parameters are variables in the function definition. Arguments are the actual values passed when calling the function.

Example: Simple Function

def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # Output: Hello, Alice!

Return Statements

The return statement sends a value back to the caller.

Example: Function with Return

def add(a, b):
    """Add two numbers"""
    return a + b

result = add(5, 3)
print(result)  # Output: 8

Default Parameters

Provide default values for parameters.

Example

def greet(name="Friend"):
    return f"Hello, {name}!"

print(greet())          # Output: Hello, Friend!
print(greet("Bob"))     # Output: Hello, Bob!

Variable Scope

Variables defined inside a function have local scope.

Example

x = 10  # Global

def my_func():
    x = 5  # Local
    print(x)  # Prints 5

my_func()
print(x)  # Prints 10

Practice Exercises

Exercise 6.1

Create a function that calculates the area of a rectangle.

Solution 
def rectangle_area(length, width):
    return length * width

print(rectangle_area(5, 10))  # Output: 50

Exercise 6.2

Write a function that checks if a number is prime.

Solution 
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True
