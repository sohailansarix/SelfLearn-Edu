Lesson 10: Advanced Topics & Next Steps

Explore advanced Python concepts and plan your learning journey beyond the basics.

Object-Oriented Programming (OOP)

Learn to organize code using classes and objects:

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, I'm {self.name}"

person = Person("Alice", 25)
print(person.greet())  # Hello, I'm Alice

List Comprehensions

Write concise and efficient list creation:

# Traditional approach
squares = []
for i in range(10):
    squares.append(i**2)

# List comprehension
squares = [i**2 for i in range(10)]

# With conditions
even_squares = [i**2 for i in range(10) if i % 2 == 0]

Decorators

Modify function behavior without changing the function itself:

def my_decorator(func):
    def wrapper():
        print("Something before function")
        func()
        print("Something after function")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
# Output:
# Something before function
# Hello!
# Something after function

Lambda Functions

Create anonymous functions for simple operations:

# Regular function
def add(x, y):
    return x + y

# Lambda equivalent
add = lambda x, y: x + y

# Useful with map, filter, sort
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x**2, numbers))
print(squared)  # [1, 4, 9, 16, 25]

Generators

Create memory-efficient iterables:

def count_up_to(n):
    i = 1
    while i <= n:
        yield i
        i += 1

for num in count_up_to(5):
    print(num)  # Prints 1, 2, 3, 4, 5
