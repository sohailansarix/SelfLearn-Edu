What is Python?

Python is a high-level, interpreted programming language known for its simple, readable syntax. It's widely used in:

  • Web Development (Django, Flask)
  • Data Science (Pandas, NumPy, Scikit-learn)
  • Automation (Scripts, bots, testing)
  • AI & Machine Learning (TensorFlow, PyTorch)
  • Desktop Applications

Step 1: Download Python

  1. Visit python.org
  2. Download Python 3.11 or later (choose your OS: Windows, macOS, or Linux)
  3. Important: Check the box "Add Python to PATH" during installation (Windows only)

Step 2: Verify Installation

Open your terminal/command prompt and type:

python --version

You should see: Python 3.11.x (or later version)

Step 3: Run Your First Program

Open terminal and type:

python
print("Hello, Python!")

You should see: Hello, Python!

Step 4: Set Up VS Code (Optional but Recommended)

  1. Download VS Code from code.visualstudio.com
  2. Install the "Python" extension by Microsoft
  3. Create a file: hello.py
  4. Type: print("Hello from VS Code!")
  5. Run: Press Ctrl+F5 or click the Run button

Practice Exercises

Exercise 1.1: Print Your Name

Write a program that prints: "My name is [Your Name]"

Solution

Exercise 1.2: Print Multiple Lines

Write a program that prints three lines with different information about you.

Solution 
print("Name: John")
print("Age: 20")
