Lesson 3: Strings & Input/Output

Learn to work with strings and get input from users.

What are Strings?

Strings are sequences of characters enclosed in quotes. You can use single quotes, double quotes, or triple quotes:

text = "Hello World"
name = 'Alice'
message = '''This is a
multi-line string'''

String Methods

Python provides many methods to work with strings:

text = "Hello World"
print(text.upper())                 # HELLO WORLD
print(text.lower())                 # hello world
print(text.replace("World", "Python"))  # Hello Python

Common String Methods

  • upper() - Convert to uppercase
  • lower() - Convert to lowercase
  • strip() - Remove leading/trailing whitespace
  • replace() - Replace substrings
  • split() - Split string into a list
  • join() - Join list elements into a string
  • find() - Find substring position
  • startswith() - Check if string starts with substring

Getting User Input

Use the input() function to get input from users:

name = input("Enter your name: ")
age = input("Enter your age: ")
print("Name:", name)
print("Age:", age)

F-Strings (Formatted String Literals)

Modern way to format strings in Python 3.6+:

name = "Alice"
age = 25
print(f"My name is {name} and I am {age} years old")
print(f"My age next year: {age + 1}")

String Concatenation

Combine strings using the + operator or join them:

first = "John"
last = "Doe"
full_name = first + " " + last
print(full_name)  # John Doe

String Slicing

Extract parts of a string using indexing:

text = "Python"
print(text[0])       # P (first character)
print(text[0:3])     # Pyt (first 3 characters)
print(text[-1])      # n (last character)

Key Points to Remember

  • Strings are immutable sequences of characters
  • Use single or double quotes - they work the same way
  • Use triple quotes for multi-line strings
  • F-strings are the modern way to format strings
  • The input() function always returns a string
  • Use .strip() to remove whitespace from input
  • Remember that string indexing starts at 0
