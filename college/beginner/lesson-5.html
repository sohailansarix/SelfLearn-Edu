Lesson 5: Loops - For and While

Learn to repeat code with loops

Introduction

Loops are used to execute a block of code multiple times. This is useful when you need to repeat the same action without writing the same code over and over.

The for Loop

The for loop is used to iterate over a sequence (like a list, tuple, string, or range).

for item in sequence:
    # Code to execute for each item
    print(item)

Example: Loop Through a List

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Output:

apple
banana
cherry

The range() Function

The range() function generates a sequence of numbers, commonly used with loops.

for i in range(5):
    print(i)

Example: range() with Different Parameters

# range(5) generates 0, 1, 2, 3, 4
for i in range(5):
    print(i)

# range(2, 8) generates 2, 3, 4, 5, 6, 7
for i in range(2, 8):
    print(i)

# range(0, 10, 2) generates 0, 2, 4, 6, 8
for i in range(0, 10, 2):
    print(i)

The while Loop

The while loop executes a block of code as long as a condition is True.

while condition:
    # Code to execute while condition is True
    print("Still looping")

Example: Simple while Loop

count = 1
while count <= 5:
    print(count)
    count = count + 1

Output: 1 2 3 4 5

Loop Control: break and continue

You can control loop execution with break and continue statements.

The break Statement

The break statement exits the loop immediately.

for i in range(10):
    if i == 5:
        break
    print(i)

Output: 0 1 2 3 4

The continue Statement

The continue statement skips the current iteration and continues with the next one.

for i in range(5):
    if i == 2:
        continue
    print(i)

Output: 0 1 3 4

Nested Loops

You can nest loops inside each other for more complex iterations.

for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

Example: Nested Loops

for i in range(3):
    for j in range(3):
        print(f"({i},{j})", end=" ")
    print()  # New line

Loop else Clause

Both for and while loops can have an else clause that executes when the loop completes normally (not with break).

for i in range(5):
    if i == 10:
        break
    print(i)
else:
    print("Loop completed normally")

Common Loop Patterns

Sum Elements in a List

numbers = [1, 2, 3, 4, 5]
total = 0
for num in numbers:
    total = total + num
print(f"Sum: {total}")

Find Maximum Value

numbers = [3, 7, 2, 9, 1]
max_value = numbers[0]
for num in numbers:
    if num > max_value:
        max_value = num
print(f"Maximum: {max_value}")

Key Points to Remember

  • for loops are used when you know how many times to iterate
  • while loops are used when the condition determines when to stop
  • Use break to exit a loop early
  • Use continue to skip to the next iteration
  • The range() function is useful for generating sequences
  • You can nest loops for complex iterations
