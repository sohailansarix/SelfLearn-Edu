Loops - For and While

25 minutes • Beginner

Learning Objectives

  • Understand for loops and iteration
  • Use while loops for conditional repetition
  • Control loops with break and continue
  • Understand nested loops

The for Loop

The for loop is used to iterate over sequences (lists, tuples, strings, ranges).

Basic Syntax

for item in sequence:
    # code executed for each item
    print(item)

Example: Loop Through a List

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Output:
apple
banana
cherry

The range() Function

The range() function generates a sequence of numbers.

Examples

# range(5) produces 0, 1, 2, 3, 4
for i in range(5):
    print(i)

# range(2, 8) produces 2, 3, 4, 5, 6, 7
for i in range(2, 8):
    print(i)

# range(0, 10, 2) produces 0, 2, 4, 6, 8
for i in range(0, 10, 2):
    print(i)

The while Loop

The while loop executes code as long as a condition is True.

Basic Syntax

while condition:
    # code executes while condition is True
    # must update condition to avoid infinite loop
    print("Still looping")

Example: Simple while Loop

count = 1
while count <= 5:
    print(count)
    count = count + 1

Output: 1 2 3 4 5

Loop Control: break and continue

break Statement

Exits the loop immediately.

for i in range(10):
    if i == 5:
        break
    print(i)

Output: 0 1 2 3 4

continue Statement

Skips the current iteration and continues to the next.

for i in range(5):
    if i == 2:
        continue
    print(i)

Output: 0 1 3 4

Nested Loops

Loops inside loops for complex iterations.

Example: Multiplication Table

for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}")

Practice Exercises

Exercise 5.1

Print numbers from 1 to 10 using a for loop.

Solution 
for i in range(1, 11):
    print(i)

Exercise 5.2

Find the sum of numbers from 1 to 100 using a while loop.

Solution 
num = 1
total = 0
while num <= 100:
    total += num
    num += 1
print(f"Sum: {total}")
